# Ejercicio-IA-con-Python---Sistema-de-Inventario
Aplicación de escritorio en Python para gestión de inventarios con interfaz gráfica desarrollada en Tkinter y almacenamiento en SQLite. Permite realizar operaciones CRUD sobre productos (nombre, precio en COP, cantidad y stock mínimo) e incluye una función de alerta que identifica productos con inventario por debajo del nivel mínimo.
